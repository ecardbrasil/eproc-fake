Este ambiente é uma simulação para fins exclusivamente educacionais, sem vínculo com o sistema oficial do Poder Judiciário (eproc).
Dicas para usar a página de Processo:
  • Utilize a busca para localizar rapidamente um processo pelo número ou partes envolvidas.
  • Os detalhes do processo ficam disponíveis ao clicar no número do processo.
  • Para visualizar documentos, clique no ícone correspondente ao lado de cada movimentação.
  • Use os filtros para refinar a lista de processos exibidos.
  • Em caso de dúvidas, consulte o manual ou entre em contato com o suporte.
